CommTone.exe 串口调试精灵 使用说明 
--------------------------------------------------------------------------------
1、串口打开后，可随时修改通讯参数，只需要按一下应用。

2、输入HEX: 用户输入数据看作16进制字节，不区分大小写，接收的会显示为16进制HEX格式。

3、输入ASC: 用户输入数据看作ASCII字符，接收的也会显示为ASCII字符。

4、自动换行: 是否在接收与发送的数据之间自动换行显示。

5、DTR/RTS: 选中后相应针变高电平，用于一些无源485/422转换器。

6、选中“回车发送”，输入区内按回车相当于按发送按钮， 如想输入多行，可用Ctrl+回车；
  不选中，则输入区内可用回车分行。

7、CRC校验，加入了16位CRC校码放到每次发送的字节数的最后两位。

8、用户输入（从串口输出）的数据显示为绿色，从串口输入的数据显示为蓝色。

9、文件发送：点击【发送文件】按钮，选择要发送的文件，发送端和另一端会自动弹出接收信
  息框。注意，发送文件时，周期的毫秒数为发送时间间隔，周期越小发送越快，如果太小可
  能另一端还没有接收完数据缓冲区已满会造成丢包现像，建议周期值20-50间为好。





CommToneX.ocx控件
--------------------
 
        CommToneX.ocx  为ActivexForm控件,可以嵌入网页中操作串口通信，
功能齐全和CommTone.exe一样, 它和CommTone.exe是相互独立的，不产生依赖关系。



CommToneX.ocx 注册方法 
---------------------
1.打开Reg.bat注册OCX组件。

2.用IE浏览器打开CommToneX.htm。

3.IE浏览器会提示阻止 ActiveX控件，确定运行。

4.打开UnReg.bat反注册组件，删除。



CommToneX.ocx 注意
---------------------
1.360等安全软件会拦截IE浏览器加载OCX，建议打开前先关闭或加入白名单。

2.如果是普通户需要管理员权限注册OCX控件。

3.CommToneX.ocx为试用版。

4.支持IE5.0或以上浏览器。

5.互联网发布时，客户端 Internet选项->安全->Internet->ActiveX项,相应的选项要启用。




其他
---------------------
CommToneX.ocx 为ActiveX Form版本的OCX,可以在IE浏览器中加载打开调试串口。
